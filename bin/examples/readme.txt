Simple Nifty Tutorial
=====================

Find the full tutorial here:
http://jmonkeyengine.org/wiki/doku.php/jme3:advanced:nifty_gui

0. Templates

Files used: 
* assets/Interface/tutorial/niftytemplate.xml (Minimal example of Nifty's XML syntax)
* src/tutorial/NiftyTemplate.java (Minimal example of Nifty's Java syntax)

1. Lay Out the Nifty GUI Screens

Files used: 
* assets/Interface/tutorial/screen1.xml (XML layouts of two screens)
* src/tutorial/Screen1.java (Java syntax equivalent to screen1.xml)

2. Add Content to Nifty GUI

Files used: 
* assets/Interface/tutorial/*.PNG (images used)
* assets/Interface/tutorial/screen2.xml (two screens with panels)
* src/tutorial/Main2.java (this Java file starts the static(!) XML Screen)
* src/tutorial/Screen2.java (Java syntax equivalent to screen2.xml)

2. Make Nifty GUI Interactive

Files used: 
* assets/Interface/tutorial/*.PNG (images used)
* assets/Interface/tutorial/screen3.xml (two screens with interactive panels)
* src/tutorial/Main3.java (this Java file starts the XML Screen)
* src/tutorial/MyStartScreen.java (the implementation of the interactive parts)
* src/tutorial/Screen3.java (Java syntax equivalent to screen3.xml)
